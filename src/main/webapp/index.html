Welcome to my exam backend

Endpoints:

User:

/api/matches/all gets a list of matches (user has to be logged in)

/api/matches/{id}/play gets a list of matches by player id(user has to be logged in)

/api/matches/{id}/location gets a list of matches by location id(user has to be logged in)

/api/matches/locations gets a list of matches all locations (user has to be logged in)

/api/matches/players gets a list of matches all players (user has to be logged in)



Admin:

/api/admin/location create a location (admin has to be logged in)

/api/admin/match create a match (admin has to be logged in)

/api/admin/player create a player (admin has to be logged in)

/api/admin/{id}/match update a match (admin has to be logged in)

/api/admin/{mid}/connect/{lid} connect a match to a location (admin has to be logged in)

/api/admin/{id}/delete delete a player (admin has to be logged in)
